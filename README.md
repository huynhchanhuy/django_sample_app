# Django Sample App
<h3>Required:</h3>
 - Django version 1.8
 <br />
 - Crispy-Form
 <br />
 - Virtualenv
 <br />
 - Pillow
 <br />
 - pytz
 <br />
 - Django Registration Redux
<br />
 - Django Contrib Comment
<h3>How to run the server ?</h3>
 - In source folder
  <br />
	source bin/activate
 - Next, go to instagram_clone folder
  <br />
 	./manage.py runserver
 - Open the browser and type "http://127.0.0.1:8000"
