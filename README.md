# Django Sample App
Required:
 - Django version 1.8
 <br />
 - Crispy-Form
 <br />
 - Virtualenv
 <br />
How to run the server ?
 - In source folder
  <br />
	source bin/activate
 - Next, go to instagram_clone folder
  <br />
 	./manage.py runserver
 - Open the browser and type "http://127.0.0.1:8000"
